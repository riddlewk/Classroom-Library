# Product vision

## Overview
Build a web-based classroom library app that:
- Identifies books by ISBN
- Auto-fetches metadata from public ISBN/book APIs
- Auto-suggests tags/categories based on content
- Stores all data in a free Supabase (Postgres) backend
- Provides a simple, teacher-friendly UI for managing a classroom collection

## Primary users
- **Teacher**: manages the library, adds/edits books, curates tags.
- **(Optional later)** Students: browse/search books, see tags and reading levels.

## Core value
- Reduce manual data entry for teachers.
- Provide consistent tagging and categorization for instructional planning.
